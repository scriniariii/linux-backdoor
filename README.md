# linux-backdoor
simple backdoor/reverse shell made in c

## Features

- Implementation of sockets in C.
- Remote execution of commands on the client.
- Function to change directory (`cd`) from the server.
- Lightweight and efficient code, compileable in Linux/Unix environments.
## Use

1. **Compile code:**
   ```bash
   gcc file.c -o file
   ```

2. **Execute:**
   ```bash
   ./file
   ```
